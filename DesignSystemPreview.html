Brand Guidelines

Understanding current design trends and user experience knowledge, we selected a bright and light design style. Our team conducted additional research to make sure the colors and visual components will stand out for a user.

Elevation & Shadows

Three elevation levels: Flat, Raised, and Overlay

Level 0 — Flat
shadow: none
Use for: Table rows, list items, sections
Level 2 — Raised
shadow: 0 2px 8px rgba(0, 0, 0, 0.06)
Use for: Cards, panels, dropdown menus, popovers
Level 4 — Overlay
shadow: 0 16px 48px rgba(0, 0, 0, 0.12)
Use for: Modals, dialogs, full-screen overlays

Z-Index scale

10-level system for consistent layering

Level Value Use case
base 0 Default stacking (most elements)
dropdown 10 Dropdowns, tooltips
sticky 20 Sticky headers, floating action buttons
fixed 30 Fixed navigation, sidebars
backdrop 40 Modal backdrops, drawer overlays
modal 50 Modals, dialogs, sheets
popover 60 Popovers that appear over modals
toast 70 Toast notifications (always visible)
tooltip 80 Tooltips (highest UI element)
debug 90 Development tools, debug overlays
